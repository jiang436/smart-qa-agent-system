# 小米米家扫拖机器人 智能家居联动

## 小爱同学语音控制

### 基础指令

| 语音指令 | 操作 |
|---------|------|
| "小爱同学，扫地" / "开始清扫" | 启动全屋清扫 |
| "小爱同学，暂停清扫" / "停止清扫" | 暂停/停止当前任务 |
| "小爱同学，让扫地机回去充电" / "回充" | 返回充电座 |
| "小爱同学，扫地机强力模式" / "安静模式" | 切换吸力档位 |
| "小爱同学，扫地机开始拖地" | 启动拖地模式 |

### 定点清扫指令

| 语音指令 | 效果 |
|---------|------|
| "小爱同学，让扫地机清扫客厅" | 指定房间清扫 |
| "小爱同学，让扫地机打扫厨房和餐厅" | 多房间连续清扫 |
| "小爱同学，让扫地机扫一下沙发前面" | 区域清扫 |

> **前提**：须在米家 APP 中完成建图 → 房间划分 → 中文命名。名称需与语音指令一字不差。

### 状态查询指令

| 语音指令 | 返回信息 |
|---------|---------|
| "小爱同学，扫地机还有多少电？" | 当前电量百分比 |
| "小爱同学，扫地机扫完了吗？" | 清扫进度/是否完成 |
| "小爱同学，扫地机在哪里？" | 部分机型支持查找机器人（发出提示音） |
| "小爱同学，扫地机滤网该换了吗？" | 耗材剩余寿命（部分新机型） |

### 使用注意事项

- 小爱音箱距机器人 5 米内唤醒效果最佳
- 背景噪音建议 <55dB（正常交谈音量）
- 多台小爱设备时在 APP 中指定默认控制音箱，避免重复响应
- 定期升级固件，确保新指令集生效（如"沿边清扫""深度拖地"）
- 语音响应延迟 <1.2 秒（2.4GHz WiFi 下）

## 米家 APP 自动化场景

### 离家自动清扫

| 触发方式 | 配置 |
|---------|------|
| 米家门锁上提反锁 | 门锁 → 扫地机：开始全屋清扫 |
| 米家人体传感器检测无人 | 所有人离开 5 分钟后 → 开始清扫 |
| 手机 GPS 离家和电子围栏 | 最后一人离家 → 开始清扫 |

### 回家暂停

| 触发方式 | 配置 |
|---------|------|
| 门锁开锁 | 门锁 → 扫地机：暂停并回充 |
| 人体传感器检测有人 | 有人回家 → 扫地机停止工作 |

### 定时清扫

- 设置路径：米家 APP → 扫地机 → 定时清扫 → 添加
- 支持：每天 / 每周几 / 自定义时间段
- 可设置勿扰时段（如 22:00-08:00 不执行定时任务）
- 建议设在外出时段（如工作日 10:00），避免噪音影响

### 晚安模式（一键场景）

触发词："小爱同学，晚安"

| 设备 | 动作 |
|------|------|
| 全屋灯光 | 全部关闭 |
| 窗帘 | 关闭 |
| 空调 | 调至睡眠模式 26℃ |
| 扫地机 | 回充（如在工作中） |
| 空气净化器 | 切换至睡眠模式 |

### 环境联动

| 触发条件 | 动作 |
|---------|------|
| 温湿度计检测 PM2.5 超标 | 小爱播报提醒关窗 + 扫地机待机（避免扬尘） |
| 水浸传感器检测到漏水 | 扫地机停止拖地 + APP 推送告警 |
| 烟雾报警器触发 | 扫地机立即停止 + 所有灯光开启 |

## 小米澎湃 OS 生态

- 小米手机可在控制中心直接查看扫地机状态和电量
- 小米电视在清扫完成时弹窗通知
- 小米手环/手表在清扫完成时震动提醒
- 所有米家设备（灯、空调、窗帘、门锁、摄像头、净化器等）均可与扫地机联动

## Apple Home 接入（Matter 协议）

- 需 iOS 16.4+ 和 HomePod/Apple TV 作为家居中枢
- 支持 Siri 控制："Hey Siri, start vacuuming" / "Hey Siri, send the vacuum home"
- 在 Apple Home APP 中查看清扫状态
- 可与 Apple Home 中的其他设备联动（如门锁、传感器）
