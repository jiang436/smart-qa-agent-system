# 扫地机器人售后保修基础知识

## 行业通用保修标准

| 部件 | 典型保修期 | 备注 |
|------|---------|------|
| 整机（含基站） | 1-2 年 | 各品牌不同 |
| 电池 | 6-12 个月 | 续航低于标称 50% 可免费更换 |
| 电机（风机/轮毂电机） | 2-3 年 | 部分品牌高端机型 |
| 激光雷达模块 | 2-3 年 | 部分品牌高端机型 |
| 边刷 / 滚刷 / 滤网 / 拖布 / 集尘袋 | 不保修 | 属于易损耗材，需自费更换 |
| 上下水套件 | 1 年 | 从安装日期起算 |

## 退换货规则

| 时间节点 | 政策 |
|---------|------|
| 签收 7 天内 | 无理由退货。要求：包装完整、配件齐全、机器未使用 |
| 签收 15 天内 | 出现质量问题可换新（同型号） |
| 超过 15 天 | 不换新，走正常保修流程 |

## 保修需要什么凭证

- **电子发票**（最优）：各品牌官方商城 / 天猫 / 京东订单记录即可
- **纸质发票**：同样有效
- **无发票**：按机器出厂日期 + 3 个月估算保修起始日（各品牌规则略有差异）

## 不保修的情况

以下情况即使在保修期内也需自费维修：
1. 人为损坏（摔落、进水、碾压宠物粪便等）
2. 私自拆卸改装
3. 使用非原厂清洁液导致水箱/管路腐蚀（维修人员可检测到化学残留）
4. 自然灾害（水灾、火灾等不可抗力）
5. 耗材自然磨损（边刷/滚刷/滤网/拖布的正常消耗）

## 售后渠道

| 渠道 | 说明 |
|------|------|
| 品牌官方 APP | 小米商城 → 我的 → 售后服务 → 申请维修 |
| 客服电话 | 各品牌 400 热线（见具体品牌页面） |
| 线下门店 | 品牌直营店 / 授权服务中心 |
| 寄修服务 | APP 内申请，快递免费上门取件（保内免运费） |
| 电商平台 | 天猫/京东旗舰店订单可在平台内申请售后 |

## 上门服务说明

- **安装上下水**：购机含安装服务的，首次免费。后续二次安装收费 100-200 元。
- **上门维修**：保内免费（含配件和人工）。保外上门费 50 元 + 配件费。
- **覆盖范围**：通常覆盖全国县级以上城市。偏远地区可能仅支持寄修。

## 延保服务

- 部分品牌提供付费延保（¥99-199/年）
- 延保需在保修期内购买
- 延保通常不覆盖电池和耗材
