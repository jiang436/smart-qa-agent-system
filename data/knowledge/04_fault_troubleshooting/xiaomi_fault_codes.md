# 小米米家扫拖机器人 故障码对照表

> 数据来源：mi.com/hk/support + mi.com/tw/support 官方帮助中心

## 通用故障码

| 错误码 | 含义 | 解决方法 |
|--------|------|---------|
| 错误 01 | 左轮过载或卡死 | 检查左轮是否有异物缠绕，手动转动确认是否顺滑 |
| 错误 02 | 右轮过载或卡死 | 检查右轮，清除缠绕物 |
| 错误 03 | 悬崖传感器异常 | 将机器移至水平地面，用干布擦拭机身底部悬崖传感器 |
| 错误 04 | 电量不足 | 放回充电座，充满电后再使用 |
| 错误 05 | 保险杠（前撞）卡住 | 轻拍保险杠测试回弹，清除卡住的异物 |
| 错误 06 | 主刷堵转 | 拆出主刷，清理缠绕毛发和两端轴承，确保转动无阻 |
| 错误 07 | 边刷堵转 | 拆下边刷，清理缠绕毛发线头后重新装回 |
| 错误 08 | 风机异常 | 重启机器人；若仍报错，联系售后 |
| 错误 09 | 尘盒或水箱未检测到 | 重新安装尘盒/水箱，确保卡扣到位，检查底部金属触点是否脏污 |
| 错误 10 | 充电异常 | 用干布擦拭充电座和机器人底部充电触点 |
| 错误 11 | 水量不足 | 取下水箱重新加满水（仅限扫拖一体机型） |
| 错误 12 | 机身倾斜或地面不平 | 将机器人移至水平表面重新启动 |

## 激光导航机型专属故障码

| App 提示 | 含义 | 解决方法 |
|---------|------|---------|
| 错误 1（激光测距传感器异常） | LDS 激光头被遮挡或不转动 | 移除顶部遮挡物，检查激光头是否自由转动，重启机器 |
| 错误 2（请将机器人放在水平地面） | 一侧驱动轮离地 | 搬到平地重启 |
| 错误 3（悬崖传感器 / 轮子悬空） | 底部传感器脏污或轮子悬空 | 搬到平地，清洁底部传感器 |
| 错误 4（主刷问题） | 主刷卡异物 | 取下主刷，清洁刷毛和轴承 |
| 错误 7（清除障碍物） | 机器人被困住无法移动 | 移开周围障碍物或搬到空旷处重启 |
| 错误 9（强磁场） | 离虚拟墙或磁条太近 | 移离虚拟墙/磁条区域后重启 |
| 错误 11（电池温度） | 电池过冷或过热 | 等待温度恢复正常后再使用（适宜温度 0-40℃） |
| 错误 12（虚拟墙 / 禁区） | 检测到禁区或虚拟墙 | 移出禁区范围后重启 |
| 错误 13（视觉导航传感器） | 视觉传感器镜头脏污 | 用软布擦拭前端视觉镜头 |
| 错误 14（内部错误） | 系统内部错误 | 重启机器，若仍无效→恢复出厂设置→联系售后 |

## 指示灯告警

| 指示灯状态 | 含义 | 处理步骤 |
|-----------|------|---------|
| 红灯常亮 | 激光头遮挡 / 主刷堵转 / 充电触点氧化 | 1.检查尘盒是否装好 2.移除地面反光物体 3.打开米家 APP 查看详细告警信息 |
| 红灯快闪 | 固件校验失败或系统严重错误 | 长按电源键 5 秒强制关机，静置 30 秒后重启 |
| 橙色灯 | 电量过低（<15%） | 立即放回充电座 |
| 白灯呼吸 | 正在充电中 | 正常状态 |
| 白灯常亮 | 充电完成或待机 | 正常状态 |

## 硬复位与重置操作

### 强制重启
长按机身电源键 5 秒以上，听到"滴"声后松手，指示灯熄灭表示已关机。静置 30 秒后短按电源键重新开机。此操作不会清除地图和配网信息。

### 恢复出厂设置
米家 APP → 设备 → 右上角"…" → 恢复出厂设置。要求电量 >30%。此操作会清除所有地图、配网和设置。

### WiFi 重置
同时按住"回充键"和"电源键"5 秒，听到语音提示"WiFi 已重置"后松手。之后需在米家 APP 中重新配网。

## 配网故障排查

| 问题 | 原因 | 解决 |
|------|------|------|
| 搜索不到设备 | 手机蓝牙未开启 | 开启蓝牙后重新搜索 |
| 配网失败 | WiFi 是 5GHz 频段 | 机器人仅支持 2.4GHz WiFi |
| 反复掉线 | 路由器开启了双频合一 | 在路由器后台关闭双频合一，2.4G 和 5G 使用不同的 SSID |
| 密码错误提示 | WiFi 密码含中文或特殊符号 | 修改密码为纯英文+数字 |
| 始终连不上 | 路由器 AP 隔离功能开启 | 在路由器后台关闭 AP 隔离 |

## 高频故障自助排查

### 地图丢失或偏移
1. 检查激光雷达罩是否有灰尘或水渍 → 用干布擦拭
2. 确认未手动搬动机器人（搬运会导致定位丢失）
3. 在 APP 中手动切换到正确的楼层地图
4. 重新执行"快速建图"（约 10 分钟）
5. 如果频繁发生→升级固件→恢复出厂设置→联系售后

### 回充失败
1. 充电座指示灯是否亮？不亮→检查电源适配器和插座
2. 充电座前后左右是否有障碍物？（需各留 0.5m 空间）
3. 充电座是否放在阳光直射或强反光地面？（会干扰红外回充信号）
4. 用干布或橡皮擦清洁机器底部充电触片和基站充电触点
5. 把机器搬到充电座正前方 1 米处，在 APP 中点"回充"
6. 以上均无效→手动将机器靠入充电座→APP 点"充电"→自动重置回充定位

### 拖地不出水
1. 清水箱是否有水？→加水至 MAX 线
2. 拖布是否过脏导致渗透不了？→更换干净拖布
3. APP 水量是否误设为最低档？→调至 2 档或 3 档
4. 出水泵是否堵塞？→用细针头轻轻疏通出水孔（不要用力捅）
5. 以上均无效→联系售后检测水泵

### 噪音突然变大
1. 检查滚刷是否缠绕头发或线缆→取出滚刷清理
2. 检查边刷是否打到硬物→检查边刷附近是否有异物
3. 检查风机口是否吸入异物→打开尘盒检查气路
4. 检查万向轮是否卡入异物→用镊子取出

### 续航断崖下降
- 新机（<6 个月）：通常是软件问题→升级固件→恢复出厂设置→若无效走售后换电池
- 旧机（>1 年）：电池自然老化→联系售后更换（¥199-299）
- 冬季低温（<5℃）：锂电池活性降低属正常现象，温度回升后续航恢复
- 长期未充电导致过放：充满电后静置 12 小时让电池管理系统重新校准
