# X30 Pro 智能扫地机器人 — 用户手册

## 产品概述

X30 Pro 是旗舰级智能扫拖一体机器人，支持扫拖同步、激光导航、3D结构光避障。

### 核心参数
- 尺寸: 350 × 350 × 97 mm
- 重量: 3.8 kg
- 电池: 5200mAh 锂电池
- 续航: 约150分钟（标准模式）
- 充电时间: 约4-5小时
- 尘盒容量: 450 ml
- 水箱容量: 250 ml
- 噪音: ≤65 dB(A)
- 越障高度: ≤20 mm
- 吸力: 4000Pa（强力模式）
- 电压: 220V-240V / 50-60Hz，输出 20V/1.2A

## 快速开始

### 开箱安装
1. 将充电座靠墙放置，左右各留出0.5米空间，前方留出1.5米空间
2. 安装边刷：将边刷对准卡槽，按下直到听到"咔哒"声
3. 如需拖地功能，安装水箱和拖布支架
4. 长按开机键3秒启动设备
5. 将设备放回充电座，首次使用建议充满电（约4-5小时）

### APP 绑定
1. 下载官方App：扫描包装盒上的二维码，或搜索"智能生活"下载
2. 注册账号并登录
3. 在App首页点击"添加设备"，选择"扫地机器人"
4. 确认设备已开机，Wi-Fi指示灯闪烁（配对模式）
5. 选择2.4G Wi-Fi并输入密码（不支持5G频段）
6. 等待连接完成（约30秒）
7. 绑定成功后可以为设备命名

### 首次清扫
1. 首次清扫建议选择"全屋清扫"模式
2. 设备会自动扫描房间，建立地图
3. 首次建图可能需要20-40分钟，期间请保持房门打开
4. 建图完成后，地图自动保存在App中
5. 可在App中设置虚拟墙和清扫禁区

## 清扫模式

### 安静模式
- 低噪音运行，适合夜间使用
- 吸力适中，适合日常维护清洁
- 续航最长约150分钟

### 标准模式
- 平衡清洁力和噪音
- 适合大多数家庭日常清洁
- 续航约120分钟

### 强力模式
- 最大吸力4000Pa，适合深度清洁
- 检测到地毯自动切换强力模式（地毯增压）
- 续航约90分钟

### 拖地模式
- 需安装水箱和拖布
- 三档水量调节：低/中/高
- 支持扫拖一体（边扫边拖）
- 拖地结束后建议清空水箱，晾干拖布

## 地图管理

### 快速建图
- 在App中点击"快速建图"
- 设备将快速扫描房间布局，不进行清扫
- 建图速度约为清扫速度的3倍

### 多楼层支持
- 最多保存3张地图
- 搬到新楼层后，设备会自动识别并切换地图
- 首次在新楼层使用建议先建图

### 虚拟墙和禁区
- 在App地图上画线设置虚拟墙（设备不会越过）
- 画框设置清扫禁区（如宠物区域、儿童玩具区）
- 支持定时生效

## 定时清扫

### 设置定时任务
1. 打开App → 设备 → 定时清扫
2. 点击"添加定时"
3. 选择时间（时:分）
4. 选择清扫模式（安静/标准/强力）
5. 选择清扫房间（全屋/指定房间）
6. 选择重复（一次/每天/工作日/周末/自定义）
7. 保存即可

### 断点续扫
- 电量低于15%时自动回充
- 充满后自动回到断点位置继续清扫
- 大户型清扫时非常实用

## Wi-Fi 连接

### 重新配网
1. 长按设备上的Wi-Fi键5秒，指示灯开始快闪
2. 在App中进入设备 → 设置 → Wi-Fi设置
3. 选择新的Wi-Fi网络并输入密码
4. 等待连接完成

### 常见Wi-Fi问题
- 仅支持2.4G频段，不支持5G Wi-Fi
- Wi-Fi名称和密码不要使用中文或特殊字符
- 路由器建议开启DHCP
- 如果连接失败，尝试重启路由器和设备

## 维护保养

### 日常保养
- 每次清扫后清空尘盒
- 每周用干布清洁充电触点和传感器
- 每次拖地后清洗拖布并晾干

### 定期维护
- 每周清洗滤网（HEPA滤网可用水冲洗，需完全晾干后装回）
- 每月清理边刷和主刷上的缠绕物（头发、线缆等）
- 每3个月更换边刷
- 每3-4个月更换HEPA滤网
- 每6个月检查主刷磨损情况

### 长时间不用
- 充满电后关机存放
- 每3个月补充充电一次
- 存放在阴凉干燥处
- 远离热源和阳光直射
