# 故障排查完整指南

## 错误码速查表

### E01 — 跌落传感器异常
- 触发条件: 设备检测到悬空状态（被抱起/楼梯边缘/翻倒）
- 常见表现: 设备停止移动，语音提示"故障E01"
- 解决方案:
  1. 将设备放回平整地面
  2. 检查底部悬崖传感器是否有灰尘（3个透明圆形窗口）
  3. 用干布轻轻擦拭传感器表面
  4. 重新开机
- 如果反复出现E01: 可能是传感器硬件故障，联系售后

### E02 — 驱动轮卡住
- 触发条件: 轮子被异物缠绕、卡在地毯边缘、线缆缠绕
- 常见表现: 设备在原地不动或转圈
- 解决方案:
  1. 关闭设备电源
  2. 翻转设备，检查左右驱动轮
  3. 清除轮子上的头发、线缆、碎屑
  4. 用手转动轮子确认可以自由旋转
  5. 重新开机
- 预防: 清扫前先整理地面的线缆和小物件

### E03 — 边刷不转
- 触发条件: 边刷卡住或电机过载
- 常见表现: 边刷不转或转动异常
- 解决方案:
  1. 关闭电源
  2. 拔出边刷（捏住卡扣向上拔）
  3. 用干净布清理边刷轴
  4. 检查边刷基部是否有头发缠绕
  5. 重新安装边刷并开机
- 如果清理后仍然不转: 可能是电机故障，需要维修

### E04 — 尘盒未安装
- 触发条件: 尘盒未正确推入或松动
- 常见表现: 开机提示"请安装尘盒"
- 解决方案:
  1. 确认尘盒完全推入到位
  2. 听到"咔哒"声才表示安装到位
  3. 检查尘盒密封圈是否变形
  4. 重新安装后重启设备

### E05 — 电池过热保护
- 触发条件: 电池温度超过安全阈值（>60°C）
- 常见表现: 设备突然停止，语音提示E05
- 解决方案:
  1. 立即将设备移至阴凉通风处
  2. 关闭电源，等待15-30分钟自然冷却
  3. 检查充电触点是否干净（氧化会导致充电异常发热）
  4. 确保设备远离热源（暖气片/阳光直射/地暖）
  5. 待冷却后重新开机
- 如果频繁出现E05: 电池可能老化，联系售后检测

### E06 — 激光雷达异常
- 触发条件: 激光雷达无法正常旋转或信号被遮挡
- 常见表现: 建图失败、路线规划异常、设备转圈
- 解决方案:
  1. 检查设备顶部圆形凸起（激光雷达）是否有遮挡物
  2. 用干软布轻轻擦拭雷达表面的透明罩
  3. 确认没有压到雷达（不要在上面放东西）
  4. 重启设备后重新建图
- 预防: 定期清洁雷达表面，避免放在落灰处

### E07 — Wi-Fi 连接失败
- 触发条件: 无法连接路由器或信号太弱
- 常见表现: App显示设备离线，无法远程控制
- 解决方案:
  1. 确认路由器正常工作（手机能上网）
  2. 确认Wi-Fi是2.4G频段（不支持5G）
  3. 检查Wi-Fi名称和密码是否有特殊字符
  4. 将设备和路由器放近一些重新配网
  5. 长按Wi-Fi键5秒进入配网模式
  6. 如果多次失败，重启路由器和设备
- 注意: 更换路由器后需要重新配网

### E08 — 水箱未安装
- 触发条件: 拖地模式下检测到水箱未安装
- 常见表现: 提示"请安装水箱"
- 解决方案:
  1. 安装水箱支架
  2. 确保水箱卡扣已扣紧
  3. 如果不需要拖地功能，在App中关闭拖地模式

## 常见故障排查

### 扫地机完全不工作（不开机）
可能原因（按概率排序）:
1. 电量耗尽: 放回充电座充电1小时后再试
2. 电源键接触不良: 长按开机键10秒强制重启
3. 充电座无电: 检查充电座指示灯是否亮
4. 电池老化/故障: 联系售后更换电池
5. 主板故障: 需要专业维修

排查步骤:
1. 确认充电座指示灯亮 → 不亮: 检查充电座电源
2. 将设备放充电座 → 观察设备指示灯 → 不亮: 检查充电触点
3. 长按开机键10秒 → 有反应: 系统卡死已恢复
4. 以上都无效 → 可能是硬件故障，送修

### 清扫不干净
可能原因:
1. 边刷或主刷磨损（刷毛变短、变形）
2. 主刷有头发/线缆缠绕
3. 尘盒已满（超过80%容量影响吸力）
4. 滤网堵塞（超过3个月未更换/清洗）
5. 吸力模式过低（安静模式清洁力较弱）
6. 地面有粘性污渍（机器人不适合清洁）

排查步骤:
1. 检查App中当前吸力模式 → 调至强力模式
2. 清空尘盒，检查是否有大块垃圾堵塞吸口
3. 检查主刷和边刷是否有缠绕物 → 清理
4. 检查滤网是否需要清洗或更换
5. 检查刷毛磨损程度 → 磨损严重需更换

### 无法回充
可能原因:
1. 充电座周围有障碍物
2. 充电座位置变动
3. 充电触点脏污
4. 红外传感器被遮挡

排查步骤:
1. 确认充电座前方和两侧空间充足
2. 清洁充电座触点（金属弹片）和设备底部触点
3. 手动将设备对准充电座，确认能否充电
4. 如果能手动充电但不能自动回充 → 红外传感器问题

### 噪音异常变大
可能原因:
1. 主刷/边刷有异物缠绕（头发、线缆）
2. 滚轮卡入异物
3. 尘盒未安装到位产生共振
4. 风机进异物
5. 强力模式下噪音本就较大

排查步骤:
1. 切换至安静模式确认噪音是否正常范围
2. 检查主刷和边刷是否有缠绕物
3. 重新安装尘盒
4. 检查滚轮是否卡异物
5. 如果安静模式下也异常 → 可能是风机故障

### 地图丢失 / 建图失败
可能原因:
1. 激光雷达有遮挡
2. 在完全黑暗的环境中清扫（视觉传感器失效）
3. App/固件版本过旧
4. 移动了充电座位置

排查步骤:
1. 清洁激光雷达表面
2. 确保清扫时有基础照明（不要在全黑环境）
3. 检查App和固件是否最新版本
4. 在App中删除旧地图，重新建图
5. 建图时保持房门打开，让设备完整扫描

### 漏水问题
可能原因:
1. 水箱密封圈老化
2. 水箱加水过满
3. 拖布支架未正确安装
4. 水量设置过高（地面材质不适合高水量）

排查步骤:
1. 检查水箱密封圈是否完好
2. 加水不要超过MAX刻度线
3. 重新安装水箱和拖布支架
4. 在地毯区域关闭拖地功能
5. 降低水量设置

### App 无法连接设备
可能原因:
1. 设备离线（Wi-Fi断开）
2. App版本过旧
3. 手机和设备不在同一Wi-Fi网络
4. 服务器维护

排查步骤:
1. 检查设备Wi-Fi指示灯是否常亮
2. 确认手机已连接同一Wi-Fi
3. 更新App到最新版本
4. 尝试重新配网
5. 重启手机和设备
